# Daemon Integration

Based on [Daemon](https://github.com/danielmiessler/Daemon) by Daniel Miessler.

## What is Daemon?

Daemon is "a public API that represents you" - exposing personal information in a structured, queryable format for AI systems.

## Components

- `daemon.md` - Personal data file (markdown sections)
- MCP Server - Model Context Protocol integration

## ATLAS Integration

Optional integration for:
- Personal AI assistant configuration
- Calendar/availability sharing
- Preference management

## Setup

1. Create your `daemon.md` based on template
2. Configure MCP server (optional)
3. Enable in `atlas.config.json`
