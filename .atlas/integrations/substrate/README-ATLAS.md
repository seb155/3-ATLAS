# Substrate Integration

Based on [Substrate](https://github.com/danielmiessler/Substrate) by Daniel Miessler.

## What is Substrate?

Substrate is "Infrastructure for Human Knowledge & Progress" - a structured knowledge base with 17+ component types.

## Component Types

| Type | Purpose |
|------|---------|
| Problems | Documented challenges |
| Solutions | Proven approaches |
| Arguments | Reasoning chains |
| Claims | Verifiable assertions |
| Data-Sources | Authoritative datasets |
| People | Who's working on what |
| Organizations | Entities |
| Projects | Active initiatives |
| Values | Foundational principles |

## ATLAS Integration

The `knowledge/` directory contains vendored Substrate components.

Use for:
- Documenting project problems
- Tracking solutions
- Building evidence-based arguments
