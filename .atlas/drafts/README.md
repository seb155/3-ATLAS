# ATLAS Drafts

Ce dossier contient les **brouillons** d'agents, commands et skills en développement.

## Workflow

```
1. Créer draft ici (.atlas/drafts/)
2. Tester le draft
3. Valider avec owner
4. Déployer vers .claude/
5. Supprimer le draft
```

## Structure

```
drafts/
├── agents/      # Brouillons d'agents
├── commands/    # Brouillons de slash commands
└── skills/      # Brouillons de skills
```

## Règle

**Ne JAMAIS déployer directement vers `.claude/` sans passer par drafts et validation.**
