# /system

Acces direct au SYSTEM-ARCHITECT (bypass ATLAS).

## Usage

```text
/system              # Affiche l'etat du systeme d'agents
/system [question]   # Question/modification au systeme
```

## Actions

Invoque directement SYSTEM-ARCHITECT (opus) qui:
- Gere le systeme d'agents AI
- Communique directement avec l'utilisateur
- Peut creer/modifier/supprimer des agents
- Met a jour la documentation

## Exemple

```text
User: /system les agents sont trop lents

SYSTEM-ARCHITECT (direct):
"J'analyse le systeme...

Probleme identifie: Le DEBUGGER utilise Sonnet pour
toutes les erreurs, meme les simples.

Proposition: Ajouter un mode 'quick' (Haiku) pour
les erreurs simples.

Tu veux que j'implemente ca?"
```
